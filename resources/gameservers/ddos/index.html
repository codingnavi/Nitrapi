This endpoint gives you a DDoS history. This endpoint is currently only available in the following data centers:

  • Germany
  • Turkey
  • Argentina

/gameservers/ddos

This command returns a history of all DDoS attacks in the last 14 days.

Request:

GET /services/<id>/gameservers/ddos?access_token=<access_token>
